Logo System

Primary Identity: The Wordmark

The primary representation of the brand is the clean, typographic "Eroica" wordmark. The symbol is reserved for specific applications like app icons or decorative patterns.

Eroica

Dynamic web behavior: Sub-brands appear organically when needed.

When to use the Wordmark

The text-only "Eroica" logo is the primary identifier. Use it for:

  • Website headers and navigation
  • Official documents and letterheads
  • Marketing materials where clarity is key
  • Brand partnerships

When to use the Symbol

The graphical symbol is secondary. Use it only for:

  • App icons and favicons
  • Profile pictures (social media)
  • Decorative patterns or watermarks
  • When the wordmark is too small to be legible
